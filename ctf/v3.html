Case #1: Password Strength

100 points

The Case

Conan needs to create a strong password to protect important files about the Black Organization. The Black Organization has sophisticated password cracking tools, so the password must be very secure.

Password Requirements: - At least 12 characters long - Uppercase and lowercase letters - Numbers and special characters - Not based on dictionary words

Case #3: Crypto Mystery

300 points

The Case

Conan intercepted an encrypted message from the Black Organization. It seems to be using a simple substitution cipher where each letter is replaced with another letter.

JVUHU TLLA TL HA YLZAHBYHUA AVTVYYVD

Clue #1

This is a Caesar cipher, where each letter is shifted by a certain number of positions in the alphabet.

Clue #2

The most common shift value is 3, but this message uses a different shift.

Clue #3

Try different shift values until the message makes sense in English.

Your Task

Decrypt the message and find the name of the meeting location. Submit it as your flag.

Try a Caesar cipher with a shift of 7. The flag should be the name of the restaurant mentioned in the decrypted message.
