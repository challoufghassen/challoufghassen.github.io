GRAY_BOX TERMINAL v1.0

🌗 Gray Box Testing (Partial Access)
🔹 Pentester Knowledge: ⚠️ Limited knowledge, like user credentials or API docs.
🔹 Approach: Simulates an insider threat or an attacker with limited access.
🔹 Methods: Recon + authenticated testing, lateral movement, privilege escalation.
🔹 Challenges: Balances depth and realism, but might miss certain blind spots.
🔹 Tools: Burp Suite, SQLmap, BloodHound, Responder.
💡 Best For: Testing authenticated attacks, API security, and privilege escalation.
